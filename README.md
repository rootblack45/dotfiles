# dotfiles

Configuration files and scripts for all of the programs I use.

## Requirements

- MacOS and Linux are supported
- Git

## Usage

Clone the repository:

```console
$ git clone https://github.com/rootblack45/dotfiles.git $HOME/.dotfiles && cd $HOME/.dotfiles
```

Run the installer:

```console
$ ./install.sh
```
